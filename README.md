# Bedouin White Parties — Frontend-First Prototype

This workspace contains a frontend-first Next.js App Router prototype for the "Bedouin White Parties" cinematic event experience.

Quick start

1. Install dependencies (I recommend pnpm):

```bash
pnpm install
pnpm dev
```

2. Open `http://localhost:3000`.

Notes
- Tailwind tokens and foundation files are in `tailwind.config.ts` and `styles/globals.css`.
- Mock data lives in `/data` and is served via `/lib/api.ts`.
- This scaffold covers Phases 1 and 2 (foundation + core MVP screens). Continue with later phases incrementally.
